# 🐶 Happy Paws – Dog Seller Website

## Overview

Happy Paws is a simple and responsive front-end website developed using **HTML, CSS, and JavaScript**. The project showcases a fictional dog seller platform where users can browse available puppies and interact with the website through basic JavaScript functionality.

This project was created to practice front-end web development concepts, including page layout, styling, and user interaction.

## Features

* Responsive and clean user interface
* Attractive hero section with a call-to-action button
* Navigation bar for easy page navigation
* Dog listing cards with images, descriptions, and prices
* "View Puppies" button with a welcome message
* "Buy Now" button displaying a purchase confirmation alert
* Modern card design with hover effects and CSS styling

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run the Project

1. Download or clone the repository.
2. Place all dog images inside the `images` folder.
3. Open `dogs.html` in any modern web browser.
4. Click **View Puppies** to see the welcome message.
5. Click **Buy Now** on any dog card to see the purchase confirmation.
